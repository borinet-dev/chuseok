꿀 송편 x 50

 
 - 사용 시 최대 MP의 50%를 순간 회복시켜준다.
  재사용 시간 5분.

 

꽃 송편 x 30

 
 - 사용 시 60분 동안 자신의 최대 HP/최대 MP를 5% 증가시키고,
  사냥을 통한 습득 EXP/SP가 20% 가 증가하는 효과가 적용된다.

 

오색 설기 x 20

 
 - 사용 시 2시간 동안 습득 EXP/SP가 15% 증가한다.

 

오색 송편 x 50

 
 - 사용 시 30분 동안 공격 속성 70, 모든 방어 속성 70 증가.

 

한복 x 1

 
 - 사용 시 30분 동안 모든 기본 능력이 +1 증가하고,
  습득 EXP/SP가 10% 증가하며 한복을 착용한 외형으로 변경된다.
  추석 이벤트가 종료되면 자동으로 삭제된다.

 - 파티 허브.
